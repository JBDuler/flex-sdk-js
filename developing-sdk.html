    • Developing SDK

    Here are the essentical command-line commands you need when developing the SDK:

    Install dependencies:

    $ yarn install

    Build the package:

    $ yarn run build

    Run tests:

    $ yarn test

    Run linter:

    $ yarn run lint

    Format code (run Prettier):

    $ yarn run format

    Serve the docpress docs

    $ yarn run serve-docs

    Build the docpress docs

    $ yarn run build-docs

    Update the Github Pages

    $ yarn run build
$ cp build/sharetribe-sdk-web.js docs/sharetribe-sdk-web.js
$ yarn run build-docs
$ mv _docpress ../
$ git checkout gh-pages
$ cp -r ../_docpress/* ./
$ rm -r ../_docpress
$ // git add, commit, push
